# SwiftLot

A self-running auction inventory pipeline. Automatically scrapes configured vehicle auctions on a schedule, stores results in a local database, and surfaces them through a filterable web UI. Built on FastAPI, React, and SQLite.

## Features

- Scheduled auction scraping (no manual trigger needed)
- Vehicle inventory with photos, specs, and odometer history
- Watchlist — track vehicles by VIN across auctions
- TX state inspection history lookup

## Stack

Python / FastAPI / Playwright / SQLite · React / Vite
